# Product Overview

Spec駆動開発（Kiro-style）の学習・実践用ブログアプリケーション。
フルスタック開発のリファレンス実装として、現代的な AWS サーバーレスアーキテクチャを採用する。

## Core Capabilities

- **ブログ記事管理**: 記事の作成・編集・削除・一覧・詳細表示
- **フロントエンド SPA**: React + Vite による高速な UI
- **サーバーレス API**: Hono ベースの軽量 REST API
- **クラウドインフラ**: AWS CDK による Infrastructure as Code

## Target Use Cases

- Spec駆動開発ワークフロー（kiro-spec-*）の学習・体験
- pnpm モノレポ構成でのフルスタック TypeScript 開発パターンの習得
- AWS サーバーレス構成の設計・デプロイの練習

## Value Proposition

学習目的を明確にしたシンプルな構成で、Spec → 実装 → デプロイまでのアジャイルサイクルを体験できる。
実運用レベルのアーキテクチャ（CDK・Hono・React 19）を採用し、学習と実践の橋渡しをする。
