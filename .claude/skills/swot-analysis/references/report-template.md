# SWOT Analysis Report Template

Use this exact structure for every report. Fill each section with evidence-based content.
All output is in Japanese. Technical terms (AWS service names, framework names, etc.) 
can remain in English.

## Template Notes

**重要度 / 緊急度 / 規模感の基準**:
- 高: プロジェクトの成否に直結する
- 中: 重要だが代替手段がある、または時間的余裕がある
- 低: あると良いが必須ではない

**推定工数の基準**:
- 1日以内: 設定変更、ドキュメント更新、依存関係アップデート程度
- 1週間: 新機能の追加、テストスイートの整備程度
- 1ヶ月: アーキテクチャの一部見直し、新モジュールの開発程度
- 1四半期: 大規模リファクタリング、新サービスの立ち上げ程度

**品質チェック（出力前に確認）**:
- [ ] S/W/O/Tそれぞれ最低3項目、理想は5-7項目
- [ ] 全項目に具体的な根拠が明記されている
- [ ] クロスSWOTの全4象限に少なくとも2戦略ある
- [ ] 優先戦略トップ3は4象限を横断して選んでいる（全部SOにならない）
- [ ] 次のアクションは明日から実行可能なレベルまで具体化されている
- [ ] エグゼクティブサマリーは分析全体の本質を反映している
